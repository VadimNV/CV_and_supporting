# Python interface / wrapper (phonons)
- *combine8x8x8Ave_F_R.py* obtains relevant averaged information on atomic positions and forces from a molecular dynamics simulation.
- *phon8x8x8_AtT.py* uses this averaged info and runs ASAP (Fortran) and Phonopy (Python) software packages, to calculate vibrational properties.
- *plotPhon.py* visualisation of the vibrational properties.
- *exactQpathMaker_atT.py* an auxiliary file - not important
